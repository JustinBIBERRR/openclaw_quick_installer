## 主要改进

### 1. 新的步骤架构
- **新增欢迎页**: 品牌展示和功能预览
- **新增成功页**: 安装摘要和下一步引导
- **改进的步骤映射**: 前端展示步骤与后端状态解耦
- **恢复逻辑优化**: 支持中断恢复和状态跳转

### 2. 现代化设计系统
- **深空背景**: `#060B14` 主背景色
- **玻璃拟态**: 半透明面板和背景模糊效果
- **霓虹配色**: 
  - 主色调: `#00E5FF` (青色)
  - 成功色: `#39FF14` (霓虹绿)
- **统一圆角**: 24px 标准圆角，12px 小圆角
- **发光效果**: 按钮和状态指示器的发光阴影

### 3. 重构的页面体验

#### 欢迎页 (`Welcome.tsx`)
- OpenClaw logo 和品牌展示
- 功能预览列表
- 主要 CTA 按钮

#### 安装进度页 (`Installing.tsx`)
- 大型进度指示器
- 实时步骤状态
- 优化的日志展示
- 手动下载引导

#### 启动页 (`Launching.tsx`)
- 状态可视化
- 诊断和修复集成
- 简化的完成流程

#### 成功页 (`Success.tsx`)
- 安装摘要展示
- 下一步建议
- 进入管理控制台的明确路径

### 4. 改进的组件系统
- **StepBar**: 新的进度条设计和动画
- **TitleBar**: 更现代的窗口控件
- **统一样式**: CSS 类和 Tailwind 配置更新
